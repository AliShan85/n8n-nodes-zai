#### [v0.3.0](https://github.com/AliShan85/ZAiChatNode/compare/v0.2.1...v0.3.0)

> 27 April 2026

- Add Z.ai Image Generation node with GLM-Image and CogView-4 model support
- Model-specific size presets with 7 recommended resolutions per model
- Custom dimension validation (GLM-Image: 1024-2048px multiples of 32, CogView-4: 512-2048px multiples of 16)
- Quality control option (HD/Standard)
- User ID parameter for content safety compliance
- Binary image output with full API response metadata (URL, created timestamp, content filter)
- Update package description and keywords
- Rename Z.ai Chat Model display name to Z.ai Chat
- Update README with documentation for both nodes
